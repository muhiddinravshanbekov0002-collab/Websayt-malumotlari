# Fast Food Telegram Bot

## Overview
This is a Telegram bot designed for fast food ordering.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/muhiddinravshanbekov0002-collab/Websayt-malumotlari.git
   ```

2. **Install Dependencies**:
   Navigate to the project directory and run:
   ```bash
   pip install -r requirements.txt
   ```

3. **Configuration**:
   - Create a `config.py` file in the root directory and define your bot token:
     ```python
     TOKEN = 'YOUR_BOT_TOKEN'
     ```

4. **Running the Bot**:
   Execute the following command in your terminal:
   ```bash
   python bot.py
   ```

## Usage
- Start the Bot on Telegram:
  Search for the bot username and click "Start" to initiate the interaction.

- **Commands**:
  - `/menu`: View the available fast food items.
  - `/order`: Place an order for your chosen items.
  - `/help`: Get assistance with using the bot.
